# ChronoCheckers — Strategy Academy

**ChronoCheckers** — премиальная веб-платформа для игры в шашки с EdTech-составляющей: ИИ-тренер уровня гроссмейстера, живые подсказки во время партии, кастомизируемые темы доски и городские лидерборды Казахстана.

## Что сделано

- **Игровое ядро**: шашки против ИИ с обязательным взятием и мульти-прыжками.
- **Режимы времени**: Zen (без таймеров, по умолчанию), Blitz (3 мин), Rapid (10 мин).
- **AI Grandmaster Coach**: пост-игровой разбор партий и живые подсказки во время игры.
- **Кастомизация**: коллекция тем доски (классические и премиум).
- **Аутентификация**: Sign In / Sign Up с выбором города Казахстана для локальных рейтингов.
- **Лидерборды**: городские таблицы по регионам Казахстана.
- **Монетизация**: бесплатный тир + Pro-подписка (темы, безлимитные подсказки).
- **Backend**: Lovable Cloud — авторизация, профили, рейтинги, история партий.

## Для кого

- Casual-игроки, которые хотят расслабленно играть в шашки.
- Студенты и начинающие, которым нужен ИИ-наставник для роста.
- Региональное шашечное сообщество Казахстана — соревнование по городам.
- Платящие пользователи, ценящие премиум-дизайн и продвинутые инструменты.

## Почему это ценно

- **Learning-first** — каждый матч превращается в урок благодаря AI Coach.
- **Гибкий темп** — от расслабленного Zen до соревновательного Blitz.
- **Премиум-дизайн** — изумрудно-кремовая академическая эстетика с золотыми акцентами на единой дизайн-системе.
- **Локальное сообщество** — городские лидерборды делают соревнование личным.
- **Понятная монетизация** — бесплатный тир полностью играбелен; Pro открывает темы и безлимитные подсказки.
